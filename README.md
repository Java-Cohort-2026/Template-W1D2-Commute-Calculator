# W1D2 Assignment: The Commute Calculator

### The Scenario
You are planning a road trip. You need to write a Java program that calculates the total cost of fuel for the journey based on your car's efficiency and the current price of gas.

### The Math
1. `Gallons Needed = Distance / Miles Per Gallon`
2. `Total Cost = Gallons Needed * Price Per Gallon`

### Requirements
- You must use the provided variables in `Main.java`.
- Your final output must be a single line starting with: `Total Trip Cost: $`
- **Hint:** Pay attention to Integer Division! Since `milesPerGallon` is an `int`, how can you ensure the result doesn't lose decimal precision?
